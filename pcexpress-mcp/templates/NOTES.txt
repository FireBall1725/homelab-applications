MCP is served over SSE:
  https://pcexpress-mcp.k8s.firekatt.ca/sse   (posts to /messages/)

The rotating OAuth refresh token is persisted to:
  {{ .Values.configmap.config.data.PCEXPRESS_STATE_DIR }}

Single-instance by design: replicas=1, strategy=Recreate. The refresh token is
single-use and rotates on every refresh, so a second pod would break with
`invalid_grant`. Do not scale this up.

Health check (no auth):
  kubectl -n {{ .Release.Namespace }} port-forward svc/{{ include "common.names.fullname" . }} 8090:8090
  curl http://localhost:8090/health

Logs:
  kubectl -n {{ .Release.Namespace }} logs deploy/{{ include "common.names.fullname" . }}
